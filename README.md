# BIENVENIDO AL SISTEMA DE  ml_biologico
Objetivo principal: Implementar un sistema de aprendizaje automático (Machine Learning) aplicado a datos biológicos, con fines académicos y de demostración.

Tecnologías utilizadas:
front se trabajo con react y vite 
backend se trabajo mysql para la base de datos 
El cluster lo consume de una url publica en la cual esta conectado en 3 nodos y un nodo maestro en donde almacena la información usando librerias en tersoflow para la red python y
octave gnu en las entradas y salidas de los datos como se muestra en los ejemplo

Instalación básica:
Clonar el repositorio en tu entorno local.
Crear un proyecto 
npm create vite@latest nombre-proyecto
Selecciona React como framework.
Elige JavaScript o TypeScript según tu preferencia.
Para ejecutar el servidor de desarrollo 
npm run dev 
En windows tener laragon entorno de desarrollo que incluye varias herramientas 
tener instalado primero 
node -v 
npm -v 
Instalar un gestor de paquetes (opcional)
Si prefieres, puedes instalar Yarn o pnpm:
npm install -g yarn
npm install -g pnpm
Este paso es para publicar tu proyecto final 
Publicar en Vercel
Ingresa a https://vercel.com.
Haz clic en New Project.
Conecta tu repositorio de GitHub.
Vercel detectará automáticamente que es un proyecto Vite/React.
Configura variables de entorno si las necesitas.
Haz clic en Deploy.
Obtendrás una URL pública como:
https://nombre-proyecto.vercel.app
